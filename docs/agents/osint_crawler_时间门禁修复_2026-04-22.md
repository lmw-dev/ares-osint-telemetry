# osint_crawler 时间门禁修复记录（2026-04-22）

## 1. 问题
- 赛前映射仅按主客队名匹配 Understat，未做时间校验。
- 导致同名历史对阵被误命中（例如 `26064` 的水晶宫 vs 西汉姆联被误映射到 `22227`，实际日期为 `2024-04-21`）。

## 2. 修复
- 文件：`src/data/osint_crawler.py`
- 核心改造：
  - Understat 候选从“单 ID”改为“候选列表”。
  - 引入时间锚点（优先使用该场最早赔率快照时间）。
  - 新增按时间最近匹配 `_pick_understat_id_by_time(...)`。
  - 新增时间门禁（默认 `45` 天），超窗不映射。
  - 在 manifest 中新增：
    - `understat_date`
    - `understat_gap_days`
  - 当历史 manifest 缺少时间审计字段时，自动触发重映射修复。

## 3. 队名字典补齐
- 文件：`src/data/team_alias_map.json`
- 新增本次期次涉及缺失别名（示例）：
  - `佛罗伦萨 -> Fiorentina`
  - `毕尔巴鄂 -> Athletic Club`
  - `阿拉维斯 -> Alaves`
  - 以及多条英冠球队全称别名
